## 存储单位

* b，bit，位，比特，存放二进制数。
* B，byte，子杰，8个位是一个字节。
* KB，kilo byte
* MB，mega byte
* GB，giga byte
* TB，tera byte
* PB，peta byte
* EB，exa byte
* ZB，zetta byte
* YB，yotta byte
* BB，bronto byte
* NB，nona byte
* DB，dogga byte
* CB，corydon byte

## 存储单位换算

* 1CB=1024DB
* 1DB=1024NB
* 1NB=1024BB
* 1BB=1024YB
* 1YB=1024ZB
* 1ZB=1024EB 
* 1EB=1024PB
* 1PB=1024TB
* 1TB=1024GB
* 1GB=1024MB
* 1MB=1024KB
* 1KB=1024B
* 1B=8b
